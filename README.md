 2. Краткое описание решения

Внутренняя учебная платформа для сотрудников ПСБ:

- личный кабинет студента с курсами, заданиями и оценками;
- загрузка решений и история отправок по каждому заданию;
- визуализация прогресса обучения;
- профиль пользователя с аватаром;
- демо-режим и работа через Docker без сложной ручной настройки.

3. Технологический стек

Backend

- Python 3.11  
- FastAPI  
- PostgreSQL 16  
- SQLAlchemy  
- Alembic (миграции)

Frontend

- TypeScript  
- React + Vite  
- MUI (Material UI)  
- Framer Motion

Инфраструктура

- Docker, Docker Compose  
- Nginx (раздача фронта)  
- GitHub

4. Структура проекта

- `backend/` — код backend-сервиса (FastAPI, модели, роуты, миграции).
- `frontend/` — SPA-frontend (React, Vite, MUI, анимации).
- `alembic/` — миграции базы данных.
- `docker-compose.yml` — единый запуск db + backend + frontend.
- `README.md` — описание проекта и инструкция по запуску.
